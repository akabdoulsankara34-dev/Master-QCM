# 🍽️ Intégral Restaurant — Borne de Démonstration

Menu digital premium avec présentation audio synchronisée.

## 🚀 Déploiement Vercel (5 minutes)

### Option 1 — Via GitHub (recommandé)

1. Créez un repository GitHub public ou privé
2. Uploadez les 3 fichiers : `index.html`, `vercel.json`, `README.md`
3. Allez sur [vercel.com](https://vercel.com) → **New Project**
4. Importez votre repository GitHub
5. Laissez les réglages par défaut → **Deploy**
6. ✅ Votre borne est en ligne sur `https://integral-restaurant.vercel.app`

### Option 2 — Vercel CLI

```bash
npm install -g vercel
cd integral-borne
vercel --prod
```

### Option 3 — Drag & Drop

1. Allez sur [vercel.com/new](https://vercel.com/new)
2. Glissez-déposez le dossier `integral-borne`
3. Déployé en 30 secondes

## 📺 Mode Borne TV

Pour utiliser en mode plein écran sur une TV ou borne :

1. Ouvrez le site déployé sur le navigateur
2. Cliquez ▶️ **Lancer la démonstration**
3. Cliquez l'icône ⛶ en bas à droite pour passer en **mode plein écran**
4. La présentation de **5 minutes** se lance automatiquement avec narration audio

### Réglages recommandés pour borne TV

- **Navigateur** : Google Chrome (meilleur support Web Speech API)
- **Résolution** : 1920×1080 ou 3840×2160 (4K)
- **Volume** : Régler via le curseur de volume dans la barre de démo
- **Langue voix** : Français (automatique)
- **Boucle** : La démo propose de recommencer automatiquement
