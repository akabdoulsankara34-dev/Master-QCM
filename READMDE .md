# Maëlys Market & Tontine - Gestion des Créanciers

Application web de gestion des créanciers avec envoi de rappels WhatsApp.

## Fonctionnalités

- ✅ Ajout de créanciers (nom, montant, reste, motif)
- ✅ Liste complète des créanciers
- ✅ Recherche de créanciers
- ✅ Modification et suppression
- ✅ Envoi de rappels WhatsApp
- ✅ Statistiques en temps réel
- ✅ Sauvegarde automatique (localStorage)
- ✅ Interface responsive

## Installation sur Vercel

1. Créer un compte sur [Vercel](https://vercel.com)
2. Installer Git et créer un dépôt GitHub
3. Pousser le code sur GitHub
4. Importer le projet sur Vercel
5. Déployer automatiquement

## Transformation en APK

Pour convertir en APK :
1. Utiliser BubbleWrap ou Capacitor
2. Générer l'APK via Android Studio
3. Signer l'application
